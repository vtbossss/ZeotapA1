Welcome to the Rule Engine API

A robust, flexible rule engine for creating, combining, and evaluating conditional rules.

About

This application allows users to create complex rules using conditions and evaluate data against these rules. Built with Django and Django REST Framework, the API is designed to provide flexibility and power in rule-based data evaluation.

Available Endpoints

  • GET /api/rules/ - List all created rules.
  • POST /api/rules/create/ - Create a new rule by defining conditions.
  • POST /api/rules/combine/ - Combine existing rules to create a more complex rule.
  • POST /api/rules/evaluate/ - Evaluate data against a specific rule.

Documentation

For a detailed guide and API documentation, please visit the /api/ endpoint.
